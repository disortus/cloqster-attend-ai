# CloqsterAttendAI
 ###
CloqsterAttendAI — это интеллектуальная система автоматизированного учёта посещаемости обучающихся, основанная на технологиях искусственного интеллекта и компьютерного зрения. Проект предназначен для образовательных учреждений и направлен на повышение точности, прозрачности и эффективности контроля посещаемости и дисциплины студентов.

Система фиксирует время входа студентов в аудиторию с помощью камер видеонаблюдения и алгоритмов распознавания лиц. На основе полученных данных CloqsterAttendAI автоматически определяет статус посещения: пришёл вовремя, опоздал или отсутствует. Вся информация сохраняется в базе данных и отображается в виде наглядной статистики для преподавателей и администрации.

Одной из ключевых особенностей проекта является автоматическое уведомление преподавателя в случае отсутствия студента на занятии без уважительной причины. Это снижает необходимость ручного ведения журналов и минимизирует человеческий фактор.
